a) Before allocation of employees, make sure the respective project is created in add project details. b) Click on project module available in the employee dashboard. c) Click on Project Allocation tab. d) In project allocation , select the employees (or single employee) to be allocated from the employee drop down provided. e) Select project ID, reporting manager, percentage allocation, employee location and allocation start date and end date.(Allocation start date and end date must be between project start and end date) f) Click submit the employees will be allocated to the selected project ID. g) Modification and view of allocation details can be done in modify allocation/view allocation tabs respectively.

Only single employee allocation can be less than 100. For bulk allocation, percentage will be only 100%.
